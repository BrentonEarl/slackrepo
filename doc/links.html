Slackware package management links

Background information

  • slackware.com — pkgtool, installpkg, upgradepkg, removepkg etc.
  • Slackware Documentation Project — Management of software packages
  • Slackware Documentation Project — Installing Software
  • Slackbook Chapter 18 — Slackware Package Management

Other package management tools

  • slackpkg (distributed in Slackware ap/) is a tool for installing or upgrading packages through a network.
  • slackpkg+ is a plugin for slackpkg. It adds support for third-party repositories.
  • sbopkg is a command-line and dialog-based tool for SlackBuilds.org.
    sqg (distributed in sbopkg) generates sbopkg queuefiles.
  • sbotools provides a ports-like interface to SlackBuilds.org.
  • slpkg is a utility to help package management in Slackware.
  • asbt is a tool for managing packages in your local copy of SlackBuilds.org.
  • sbo_tools is a set of Bash scripts to help you in your daily use of SlackBuilds.org.
  • <a class=”ext” href=https://github.com/cwilling/hoorex”>hoorex</a> determines (based on REQUIRES field of SBo .info files) Who Requires a particular package.
  • slkbuild is an Arch-like wrapper script for easy Slackware packaging.
  • gen_repos_files.sh maintains metadata in a package repository, including package signing and a changelog.
  • checkpkg is a script for checking a Slackware package or a build log for defects and irregularities.
  • slacktrack (distributed in Slackware d/) is a tool to assist with building Slackware packages from <package>.build scripts.
  • Alien’s SlackBuild Toolkit is a web-based SlackBuild generator.
  • mkslack is a SlackBuild generator script.
  • QTGZManager is a Slackware pkgtools GUI and frontend.
  • sboui is an ncurses-based user interface for sbopkg and sbotools.
  • Slackroll is a package or update manager for Slackware systems.
  • usm is a unified slackware package manager that supports dependency resolution.
  • slack-utils is a handful of commands to quickly/easily access information on the Slackware Linux distribution.
  • depfinder scans Slackware packages and outputs a list of their dependencies.
  • sbbdep is a tool for exploring binary runtime dependencies on Slackware and Slackware based systems.
  • cpan2tgz (available via SBo) creates Slackware packages from the CPAN repository.
  • gem2tgz (available via SBo) creates Slackware packages from the rubygem repository.
  • piptgz (available via SBo) creates Slackware packages from the PyPi repository.
  • npm2tgz (available via SBo) creates Slackware packages from the npm repository.

Other distros

  • pkgsrc is a framework for building third-party software on NetBSD and other UNIX-like systems.
  • Salix tools include spi, sbobuild, slapt-get and gslapt, sourcery, and spkg.
  • Zenwalk GNU Linux has netpkg.
