Package

A prebuilt package can be downloaded here:

Git

You can clone the git repository, and then run the embedded SlackBuild to create a package that you can install:

git clone git@github.com:idlemoor/slackrepo.git
cd slackrepo
git archive --format=tar --prefix=slackrepo-0.2.0/ HEAD | gzip > SlackBuild/slackrepo-0.2.0.tar.gz
cd SlackBuild
VERSION=0.2.0 TAG=_github sh ./slackrepo.SlackBuild
installpkg /tmp/slackrepo-0.2.0-noarch-1_github.tgz
