Introduction

Slackrepo is a highly automated tool for Slackware users to manage their own package repositories. It builds packages from a repository of SlackBuilds (for example, a clone of SlackBuilds.org), and then outputs the packages to a package repository.

See the Quick start and Download pages for more information.

New release 0.3, September 2017

After a long time in development, here is a new release for you!

Important! The default hintfiles are now in a separate package.
Here are the two prebuilt packages that you need:
slackrepo-0.3-noarch-1_dbs.txz
slackrepo-hintfiles-20170826-noarch-1_dbs.txz
See the Download page for signatures and md5sums.

New features in this release include fine-grained dependency control (SUBSTITUTE, DELREQUIRES, BUILDTIME and RUNTIME), resource monitoring, Python 3 injection, and more lint checks with suppression of warnings. Bug fixes include --install, temp file usage and mount handling. See the News page for more details.
