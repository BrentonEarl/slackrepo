Software

  • slackpkg (distributed in Slackware ap/) is a tool for installing or upgrading packages through a network.
  • slackpkg+ is a plugin for slackpkg. It adds support for third-party repositories.
  • sbopkg is a command-line and dialog-based tool for SlackBuilds.org.
  • sqg (distributed in sbopkg) generates sbopkg queuefiles.
  • sbo_tools is a set of Bash scripts to help you in your daily use of SlackBuilds.org.
  • sbotools provides a ports-like interface to SlackBuilds.org.
  • asbt is a tool for managing packages in your local copy of SlackBuilds.org.
  • gen_repos_files.sh maintains metadata in a package repository, including package signing and a changelog.
  • checkpkg is a script for checking a Slackware package or a build log for defects and irregularities.
  • slacktrack (distributed in Slackware d/) is a tool to assist with building Slackware packages from <package>.build scripts.
  • src2pkg is a tool for creating packages from source code or other base material.
  • mkslack is a a SlackBuild generator script.
  • QTGZManager is a Slackware pkgtools GUI and frontend.
